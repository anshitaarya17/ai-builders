# Security Policies (Excerpt)

## Data Classification
- Public: approved for external sharing.
- Internal: for employees and contractors.
- Confidential: restricted to authorized teams.
- Restricted: requires executive approval and legal review.

## Access Control
- Least privilege is mandatory for all systems.
- Access reviews occur quarterly.
- Privileged access must use time-bound elevation.

## Incident Handling
- P1 incidents are acknowledged within 5 minutes.
- Notifications must include customer impact and mitigation steps.
